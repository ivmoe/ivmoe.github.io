在Ubuntu20.04部署DHCP并设置iptables NAT转发

将 Ubuntu 设置为一个路由器。

安装环境

  • Ubuntu 20.04 LTS
  • 需具备双网卡（一个网卡用来正常连接 site 内网/外网，一个做来分发 ip 做维护用）

配置

1、修改 /etc/default/isc-dhcp-server

2、给充当 LAN 口的网卡配置静态 IP 地址

3、DHCP 的本地配置文件是 /etc/dhcp/dhcpd.conf

4、启动 DHCP 服务

5、防火墙配置（若有）

iptables NAT转发

修改文件 /etc/sysctl.conf

配置 iptables
